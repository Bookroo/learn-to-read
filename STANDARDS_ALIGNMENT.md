# Bookroo Learn to Read — Common Core Alignment (ELA K–2)

**Updated July 2026** · Companion docs: `GUIDE.md` (curriculum principles), `EXERCISES.md` (exercise catalog), `MODULE_ORDER_REVIEW.md` (module scope & sequence)

**Public page:** `/courses/learn-to-read/common-core` (data: `client/src/data/learn-to-read-standards.ts` — keep in sync with this file when coverage changes).

Foundational reading skills are the core of Bookroo's Learn to Read curriculum. The tables below
map the program against the Common Core State Standards for **Reading: Foundational Skills**
(K–2), plus the **Reading: Literature** and **Language** standards the program addresses.

Unlike a typical alignment chart, every row names the *specific exercise or routine* that teaches
the standard — each one can be inspected in the public exercise directory
(`/courses/learn-to-read/exercises`) or played directly at `/reading-games`.

## How to read the coverage column

Bookroo publishes its full 9-module scope & sequence, and this document is honest about build
status: Modules 1–2 are live, Module 3 is in draft, Modules 4–9 are scoped. A school can rely on
"Taught today" rows immediately; "Planned" rows carry the module where the skill lands.

| Marker | Meaning |
|---|---|
| ✓✓ | **Taught extensively today** — explicit instruction plus repeated, spiraled practice (Modules 1–2, live) |
| ✓ | **Taught today** — live in the program, with planned expansion noted |
| ◐ | **In draft** — Module 3 lessons exist in draft form |
| ○ | **Planned** — scoped in the published Module 4–9 outline; module noted |
| — | Not currently planned |

## Beyond the standards

Alignment charts can't capture the parts of Bookroo that make the standards stick:

- **A teacher in every session.** Lessons are fully scripted for a parent or tutor — what to say
  (read aloud), what to watch for (helper-only notes), when to wait. Delivery is 1:1 by design.
- **100% decodability discipline.** Every word a child is asked to read is verified against the
  cumulative taught set by our curriculum tooling — including game distractors and book text.
  Word selection is screened against a children's-picture-book frequency lexicon.
- **Real books from Lesson 1.** Three letters are enough for a first story; every book ends with
  comprehension questions.
- **Spiral review with readiness gates.** Every module ends in a review lesson with an explicit
  parent-facing readiness check before advancing.
- **Named, kid-facing strategies.** Finger Words, Sticky Sounds, Letter Friends, Heart Words,
  Brain Words — children learn *how* to attack words, not just which words.
- **Free.** Every lesson, book, game, and printable.
